# Proyek SOLID OOP Python
**P12: Dokumentasi dan Version Control**

## Deskripsi Proyek
Proyek ini mendemonstrasikan implementasi prinsip SOLID (SRP, OCP, DIP) pada sistem checkout dan registrasi mahasiswa dengan menggunakan Abstraction (Interface) dan Dependency Injection. Kode ini juga telah dilengkapi dengan dokumentasi Docstring (Google Style) dan sistem Logging profesional.

## Struktur File
- `P12-PBO/refactor_solid.py`: Kode inti yang sudah direfaktor dan ditambahkan logging.
- `P12-PBO/latihan_mandiri.py`: Implementasi SOLID pada sistem registrasi mahasiswa.
- `README.md`: Dokumen panduan proyek ini.

## Cara Menjalankan
1. Pastikan Python 3.x terinstal di laptop kamu.
2. Buka terminal di folder proyek.
3. Jalankan file utama dengan perintah:
   ```bash
   python P12-PBO/refactor_solid.py
